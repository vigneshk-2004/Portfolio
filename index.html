Real-Time IoT Data Monitoring System Using AWS IoT Core, WebSocket, and EC2

Started with registering a "Thing" and generated the required certificates and policies. Then, I connected my local MQTT client using Python's paho-mqtt to publish and subscribe to AWS IoT Core topics. After that, I built a WebSocket server in Python that listens to MQTT messages and sends real-time data to connected web clients. I created a simple HTML and JavaScript web interface to display the data live and hosted it publicly using AWS S3 static website hosting. Finally, I deployed the WebSocket server on an AWS EC2 instance, secured everything using IAM roles and security policies, and monitored the system using AWS CloudWatch.
